# 🌍 Vaycay

Vaycay is a modern travel booking web application that enables users to explore destinations, browse travel packages, and book their next vacation with ease. The application provides a clean, responsive interface and a seamless booking experience.

## 🚀 Features

- User Authentication
- Browse Travel Destinations
- Explore Vacation Packages
- Responsive UI
- Secure Booking Flow
- Modern React Frontend
- Backend API Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/deeparastogi07/vaycay.git
```

### Navigate to the project

```bash
cd vaycay
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

### Start the development servers

Backend

```bash
npm run dev
```

Frontend

```bash
cd ../client
npm run dev
```
